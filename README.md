# Tempo Pump 🚀

> Fair Launch Platform on Tempo Chain

一个类似 Four.meme 的 meme 代币发射台，专为 Tempo 链设计。

## 特点

- 使用 USDC 支付（需 approve）
- 改进型指数曲线（初始 $4k → 毕业 $65k）
- 打满需注入 12,000 USDC
- 平台手续费 1%
- 自动毕业并创建流动性池
