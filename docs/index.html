Tempomeme on Tempo

Tempomeme is a USDC-native meme launchpad built on Tempo Mainnet. The product pairs Tempo's payments-first design with internal curve trading, a clear graduation threshold, and a defined post-curve liquidity path.

1. Chain

Tempomeme runs on Tempo Mainnet because the launch flow is expressed in stablecoin terms from start to finish. Token creation, price discovery, and graduation are all designed around a single chain and a single settlement asset.

Network Parameters

Network: Tempo Mainnet
Chain ID: 4217
RPC: https://rpc.tempo.xyz

Settlement Asset

All current platform trading is settled in USDC on Tempo, which keeps launch pricing, trade flow, and graduation accounting in stablecoin terms.
USDC: 0x20c000000000000000000000b9537d11c60e8b50

  • Users need an EVM wallet connected to Tempo Mainnet.
  • Token creation, buying, and selling all run on Tempo without switching to a separate execution environment.
  • The current launch flow uses USDC as the settlement token for the internal curve.
  • This makes the entire launch experience easier to reason about in USD terms instead of volatile gas-asset terms.

2. Price Curve and External Migration

Tempomeme uses a quadratic internal curve for price discovery. Trading stays inside the platform until 80% of supply is sold, and only then does the project graduate toward external liquidity deployment.

Primary Launch Contract

0x37B8Cce1b4aeD401A26f01B8f19f87d352Cb3ABf

Internal Sale Cap

800,000,000 tokens
This represents 80% of the total supply.

Curve Formula
price(sold) = 0.000003 + 0.000062 * (sold / 800,000,000)^2
Detailed Calculation
Let: saleCap = 800,000,000 r = sold / saleCap Then: price(sold) = 0.000003 + 0.000062 * r^2 Buy side: buyFee = usdcIn * 1% netUsdc = usdcIn - buyFee tokensOut = netUsdc / price(sold) Sell side: grossUsdcBack = tokensIn * price(sold) sellFee = grossUsdcBack * 1% usdcOut = grossUsdcBack - sellFee Migration condition: migrate when sold >= 800,000,000
  • The curve starts near 0.00000300 USDC.
  • Because the pricing model is quadratic, the price accelerates more aggressively later in the sale than at the beginning.
  • The simplified ratio form is r = sold / 800,000,000, so all price movement is driven by sold supply progress on the internal curve.
  • For buys, the 1% fee is removed first, then the remaining USDC is converted at the current curve price.
  • For sells, the token amount is first converted back into gross USDC at the current curve price, then the 1% fee is deducted.
  • External migration is triggered only after 800,000,000 tokens are sold on the internal curve.
  • On a no-sell path, completely filling the curve is about 19,124.58 USDC gross user spend.
  • After the current 1% trading fee, the contract retains about 18,933.33 USDC net.
  • At graduation, the remaining 20% of token supply and 19,124.58 USDC are intended to be added as liquidity on Uniswap V2.

3. Official Token

The official ecosystem token is $TME. Tempomeme describes the exchange fee policy around a 1% trading fee, with most of that fee routed back into the official token and the remainder reserved for the treasury wallet.

Buyback + Airdrop

90% of the 1% trading fee is allocated to buy back $TME and distribute it through airdrops.

Treasury Allocation

10% of the 1% trading fee flows into the treasury wallet.

  • The buyback portion is intended to create recurring market demand for $TME.
  • The airdrop component is intended to route part of exchange activity back to users through $TME distribution.
  • The treasury portion is reserved for long-term protocol operations and maintenance.
